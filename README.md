# Tourist Agency Course Project

Курсовой проект по дисциплине **«Технологии программирования»**.  
Тема проекта: **«Туристическое агентство»**.

## О проекте

Проект представляет собой программную систему для автоматизации деятельности туристического агентства.  
Приложение позволяет работать с данными о клиентах, турах, заявках и бронированиях, а также выполнять расчёт стоимости туристических услуг.

Курсовой проект выполнен поэтапно на основе лабораторных работ **1–7** 3 семестра.

## Цель проекта

Разработка программного приложения **«Туристическое агентство»** с использованием современных технологий программирования, тестирования, логирования, системы контроля версий и элементов параллельного программирования.

## Задачи проекта

- спроектировать структуру приложения;
- реализовать основные сущности предметной области;
- организовать обработку данных о клиентах и турах;
- реализовать оформление заявок и бронирований;
- выполнить расчёт стоимости туров;
- подключить внешнюю библиотеку логирования;
- разработать модульные тесты;
- вести проект в GitHub-репозитории;
- применить элементы параллельного программирования.

## Основные возможности

Программа реализует следующий функционал:

- управление клиентами;
- управление турами;
- оформление заявок;
- бронирование туров;
- расчёт итоговой стоимости;
- поиск, фильтрация и сортировка данных;
- логирование действий пользователя и ошибок;
- модульное тестирование;
- хранение проекта в системе контроля версий Git.

## Сборка и запуск проекта

### Запуск в Visual Studio

1. Открыть файл решения `TouristAgency.sln`.
2. Восстановить зависимости проекта.
3. Выполнить сборку решения.
4. Запустить приложение.

### Запуск через терминал

```bash
dotnet restore
dotnet build
dotnet run --project src
```

## Тестирование

Для запуска модульных тестов выполнить команду:

```bash
dotnet test
```

## Логирование

В проекте используется библиотека **log4net**, которая позволяет:

* фиксировать действия пользователя;
* записывать ошибки выполнения;
* сохранять служебную информацию о работе приложения.
